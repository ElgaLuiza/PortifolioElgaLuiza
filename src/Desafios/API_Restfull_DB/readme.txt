Projeto Convidados - Banco de Dados

* Criar um projeto seguindo as práticas, padrões e ferramentas dos outros projetos já criados
* Criar as tabelas "contatos", "convidados", "criticas". 
* Criar rotas CRUD para as tabelas criadas.
  Elas devem seguir:
	* verbos corretos
	* retornando mensagem do que a rota vai fazer
	* retornando o status de acordo com o verbo  
* Para rotas diferentes dos contatos retornar status 404 e a mensagem página não encontrada
* Criar o teste de cada rota no Insomnia
